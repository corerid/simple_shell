# simple_shell
This program acts like a shell that you can type command such as ls -a , cat file

Contributor
1. Toktak
2. Corerid

### compile with GCC
```
gcc shell.c -o shell
```

### interactive mode
run by command
```
$ ./shell
```

### batch mode
run by command
```
$ ./shell filename
```
